Detailed Description

This file declares the UNSW 2014 walk generator. It was refactored to fit into the B-Human framework. This version is the DNT implementation of the BHuman/UNSW walk 2014 generator.

Original authors: Bernhard Hengst, Thomas Rfer

Author
Thomas Groot
