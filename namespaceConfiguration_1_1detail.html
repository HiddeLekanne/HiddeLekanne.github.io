std::vector< std::string > Configuration::detail::getConfigLocations ( )

Get a list of folders to search for configuration files.

This list is in order of preference: check the folders for config file existence in front-to-back order.

Since we only copy the configurations for the target robot and a specific location (if omitted "default"), we can simply hardcode the following paths.

The folders are relative to the folder name returned by getConfigBaseDir:

  • "config/robot/"
  • "config/location/"
  • "config/default/"
  • "config/camera/"
Returns
An ordered list of folders to look for configuration files.
