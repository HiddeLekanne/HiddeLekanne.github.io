Detailed Description

Use this class to play sounds in a separate thread.

This will briefly spawn a new thread, so keep in mind the overhead of running more threads than physically available on our platform.

Because of the static member variables and functions, we dont need to construct a soundplayer object every time, but instead use the class function for playing sounds
