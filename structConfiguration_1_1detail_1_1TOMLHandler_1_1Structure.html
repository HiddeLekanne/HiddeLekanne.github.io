Detailed Description

Stores information of the TOML expression that is being processed.

It follows the same design as TOMLValue does for values. The type_ is set to any of the values, and the corresponding union field is set.

This type should not be publically accessible without some redesign! As it is, this class is easily modified to a corrupt state. As such, it should only be used within this TOMLHandler.
