void ALRobot::updateMotionSemaphore ( )
private

Updates the semaphore that locks the motionthread, releaseing the lock. This is done by udpating the value of the semaphore to 1 (regardless of the previous value). This value represents the number of times the motionthread can update before alrobot has to be updated again, and should never exceed 1. Since alrobot updates slower than the motionthread, this will also prevent simultaneous read-and-writes by both the motionthread and alrobot to shared memory.
