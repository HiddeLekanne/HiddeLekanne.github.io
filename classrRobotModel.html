Detailed Description

Provides a forward kinematic transformation between two links. Get it by specifying the orgin point and its end point in the transform function: transform(RobotFrames from, RobotFrames to)

Member Enumeration Documentation

enum rRobotModel::RobotFrames

The enumerator class to iterate through all possible connection from the NAO robot class. Contains every extremity of the robot and also the center of mass and top and bottom camera. The last element is not used as a connection and is it used only as a way of getting the number of elements in this enum

Member Data Documentation

Model rRobotModel::robot_connections_

The robot model provides transformation matrices of given connections in the robot with the torso as reference point. The representation works as an array where the index of a robot frame should be given, and it will return the tranformation in the form of a pair: (Quaternion, Translation).
