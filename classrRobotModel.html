Detailed Description

Provides a forward kinematic transformation between two links.

Get it by specifying the orgin point and its end point in the transform function: transform(RobotFrames from, RobotFrames to)

using rRobotModel::Model = std::array<std::pair<Eigen::Quaternion<float, Eigen::DontAlign>, Eigen::Translation3f>, RobotFrames::CountOfFrames>

Do not use alignment, because that all which are using this representation with dynamic allocation would need specific alignment NEW operator.

This is simply too much hassle. When you use Model connections you can always copy quaternion to local variable which is aligned variable and get a speedup that way.

◆ RobotFrames

enum rRobotModel::RobotFrames

The enumerator class to iterate through all possible connection from the NAO robot class.

Contains every extremity of the robot and also the center of mass and top and bottom camera. The last element is not used as a connection and is it used only as a way of getting the number of elements in this enum

Compute the relative position of an object to the robot by projecting the point onto the ground.

NOTE: computes the distance to the given point, which projects the pixel to the ground, if you give the centre of an object, the point will be projected on the ground behind the real object for points far away, thus the distance will be overestimated For example if you give the centre of the ball, it will overestimate the distance, since it projects that point to the ground behind the ball

Parameters
pixel_xx position in the image.
pixel_yy position in the image.
camera_settingscamera settings of the current cycle.
Returns
The relative position of the pixel on the ground plane to the robot in mm.

◆ robot_connections_

Model rRobotModel::robot_connections_

The robot model provides transformation matrices of given connections in the robot with the torso as reference point.

The representation works as an array where the index of a robot frame should be given, and it will return the tranformation in the form of a pair: (Quaternion, Translation).
