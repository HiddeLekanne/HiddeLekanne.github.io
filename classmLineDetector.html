Detailed Description

rLineDetector This module finds lines using a threshold on the y channel to filter out dark pixels combined with a threshold on green chromaticity to filter out green pixels.

It then applies a fast approximation of canny edge detection. Finally lines are found using opencv's fld (fast line detection) implementation on the canny image. Currently only works for the top camera.

Author
Wouter
