Detailed Description

mInterfaceCommunication: Constructs a TCP server on which representations are sent to a remote client.

mInterfaceCommunication: class that receives representations from the robot and converts them into actual representations which can be used in widgets for visualisations.

Author
Sébastien & Caitlin
