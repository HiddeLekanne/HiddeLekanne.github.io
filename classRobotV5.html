RobotV5::RobotV5 ( boost::shared_ptr< AL::ALBroker >  p_broker)

Constructor creates Naoqi module.

RobotV5: A Naoqi module that writes all the required values from ALMemory to shared memory.

Parameters
pBrokerALBroker that gives access to Naoqi.

Also reads commands from shared memory and writes to the DCM.

Only works for the V5 version of the robot!
