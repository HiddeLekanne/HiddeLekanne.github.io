class  mMotionSelector
 

Detailed Description

This file handles which motion is executed.

It takes as input all the outputs from all motion modules. Currently that is the interpolation engine and the naoqi walk within the motion request. Which motion is executed is decided by the requested motion by the behaviour engine in combination with the motion that is currently executed. Output currently consists of joint angles and a naoqi walk request.
