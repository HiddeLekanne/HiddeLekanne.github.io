Detailed Description

LolaVisitor: Unpacks a message with sensor values into shared memory by visiting each value one by one.

For example: first the key "Position" is found, then the updatePositionSensor method is used to parse the coming values. The next values will be placed in shared memory in the spots for the joint position values. The order is based on the indices from position_sensor_list_ (in the case of the position sensor). When a next map key is encountered, the update function will be switched and the next values will be put in the right place.
