std::string SystemTools::getRootDir ( )

Get the absolute path to the root directory of the project on the robot.

This will read the Bash variable $DNT_ROOT. If it exists and is not empty, this value will be returned. Otherwise it will be assumed the project root is at ~/DNT/.

This function caches the value after the first call. Internally, a system call is performed, which can be "slow". Because this value shouldn't change during runtime, the value is computed once and stored in a static variable. All future calls to this function will return this cached value.

Exceptions
std::runtime_errorIf there is any problem with opening or closing a POSIX pipe.
Returns
The absolute path to the root project directory.
