Detailed Description

TCP: base class for the tcp client and server, implements sending and receiving data over the tcp connection. When data is send, first a header with a known size is send, followed by the actual message. The header contains the length of the message. This way we known when we have received a full message (after size of the header and header.data_size amount of bytes).
