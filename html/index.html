DNT2017

Tested working requirements version numbers:

  • clang 3.8
  • cmake 3.5.1
  • Ubuntu 16.04.2 LTS
  • doxygen 1.8.11
  • opencv-3.3
  • qt 5.5.1

Before you try to build or copy to robots, first install some system-wide settings:

1 cd util/scripts
2 ./install_dnt

Before you copy a new executable to a fresh robot, first install our framework on the robot using:

1 cd util/scripts
2 ./install_robot <ip-address>

To copy the new executable to a robot, call:

1 robocopy <ip-address>

To view the possible command line argument for configuring the robot, call:

1 robocopy --help

To compile the documentation using doxygen, from the root project directory, call:

1 doxygen

After completion, the html or latex subfolders in docs will contain the documentation files. A useful start point is the docs/html/index.html file.
