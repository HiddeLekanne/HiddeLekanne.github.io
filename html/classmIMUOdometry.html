Detailed Description

For now this module mainly serves as a test for the robot model. TODO, will contain the following:

This modules purpose is to get an estimation of the odometry of the robot by fusing the raw gyroscope values and accelerometer per update cycles. To make it more robust a Kalman Filter will be added to smooth the values
