This method calculates the joint angles for the legs of the robot from a Pose3f for each leg.

Parameters

positionRightThe desired position (translation + rotation) of the right foots ankle point.
jointAnglesThe instance of JointAngles where the resulting joint angles are written into.
ratioThe ratio between the left and right yaw angle.
Returns
Whether the target position was reachable or not (if the given target position is not reachable the computation proceeds using the closest reachable position near the target).
