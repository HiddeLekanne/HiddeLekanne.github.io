rWalkGenerator::WalkMode rMotionRequest::walkMode

walkMode sets the type of walk to be used. This can be either one of: rWalkGenerator::walkMode::speedMode - Speed in mm/s and radians/s. rWalkGenerator::walkMode::stepSizeMode - Step size in mm and radians (robot will do exact steps). rWalkGenerator::walkMode::targetMode - UNDER DEVELOPMENT. Walk to given relative position in mm and radians.

Pose2f rMotionRequest::walkSpeed = Pose2f()

Set walkSpeed, stepSize or targetMode with: Pose2f(rotation, Eigen::Vector2f(forward, sideways)) Maximum values for rotation, forward and sideways can be set or found in the walking engine.
