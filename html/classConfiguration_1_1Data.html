Data::Value & Configuration::Data::get_entry ( const Key &  key)
private

Splits up the path and traverses data_ to find the requested key.

Keys describe a path to a specific value. Each node in the path is separated by dots. This makes a key look like "a.b.c", with "c" the leaf node, and "a" and "b" intermediate nodes.

This method performs an iterative lookup in data_ to find the key. It splits up the path into separate nodes, and iteratively goes through data_ until it has found the leaf node. It will then return a reference to the leaf node.

Exceptions
std::runtime_errorIf any part of the lookup key is empty, or if an intemediate node already stores a value.
Returns
A reference to the stored value.

Map Configuration::Data::data_
private

Storage of the parsed configuration data.

This map represents a tree: each entry's value is either a leaf node, or stores another map (if value.type() == TOMLValue::Type::table).
