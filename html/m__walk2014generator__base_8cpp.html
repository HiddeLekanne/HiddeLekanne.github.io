This file implements the UNSW 2014 walk generator. It was originally refactored to fit into the B-Human framework.

The period of each foot-step is set by T. T generates the forcing function by alternatively lifting each foot. Control: The change of support foot is driven by the ZMP switching sign, but must be > T/2. If > 3*T we switch to try to revive. The front-back sway is controlled by ankle tilts proportional to the y gyro. The user specifies forward(m), left(m), turn(radians) to activate the walk. If these values are all zero the robot stands.

Any comment in the code with generator.* or walk_generator_output->* signifies a possible output representation that can be added if requested.

Original authors: Bernhard Hengst, Thomas Röfer
