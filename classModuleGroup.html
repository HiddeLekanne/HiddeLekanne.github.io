Detailed Description

ModuleGroup: A group of modules that are updated on a separate thread.

Their updates are sorted only based on the inputs in this groups.

◆ sortModules()

void ModuleGroup::sortModules ( )

Sort module_update_function_ by update order.

Warning
This implementation expects that module_update_functions_ represents a directed acyclic graph. UpdateInformation participating in the cycle, and UpdateInformation depending on those in the cycle will be excluded from the ordering.

This implements Kahn's algorithm. UpdateInformation that have no input_representations are placed first. Then, each UpdateInformation that has only input_representations which are provided by UpdateInformation which are already stored in this sorted collection.

The order between "equal elements" (for instance, elements that all have no input_representations) is based on the order std::set encounters them.

See also
Kahn's algorithm
Topological sorting on Wikipedia
