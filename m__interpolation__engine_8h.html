class  mInterpolationEngine
 

Detailed Description

This file handles interpolation of joints for specific motions. It takes a motion request from the behaviour engine. After checking if the motion is executable in this module, the keyframe is loaded. A keyframe can be obtained directly from Choregraphe, or using the mof2keyframe converter if you have a mof file. Interpolation is done linearly from joint value to joint value.
