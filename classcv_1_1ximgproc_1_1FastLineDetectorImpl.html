virtual

Detect lines in the input image.

Parameters
_imageA grayscale(CV_8UC1) input image. If only a roi needs to be selected, use lsd_ptr->detect(image(roi), ..., lines); lines += Scalar(roi.x, roi.y, roi.x, roi.y);
_linesReturn: A vector of Vec4f elements specifying the beginning and ending point of a line. Where Vec4f is (x1, y1, x2, y2), point 1 is the start, point 2 is the end. Returned lines are directed so that the brighter side is placed on left.
