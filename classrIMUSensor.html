Detailed Description

IMUSensorRepresentation: representation that holds the values from the Inertial Measurement Unit (IMU).

Gyroscope values specify the rotational speed of the robot in radians per second relative to the center of the body. Accelerometer value specify the acceleration of the robot in metres per second squared (m/s^2) relative to the center of the body. Angles specify the orientation of the robot in radians.
